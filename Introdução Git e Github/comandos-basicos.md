#Comandos Basicos Git

```bash
#Clona um repositório GIT.
$ git clone “endereço repositório”

#Usado para criar um novo repositório GIT.
$ git init

#Usado para adicionar arquivos.
$ git add “nome do arquivo”

#Usado para confirmar as alterações.
$ git commit –m “mensagem”

#Envia as alterações feitas para o repositório remoto.
$ git push

#Mesclar todas as alterações presentes no repositório remoto para o diretório de trabalho local.
$ git pull

```
